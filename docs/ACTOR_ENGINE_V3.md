# 🚀 محرك قرار الفاعل V3 - Actor Decision Engine

## 📋 المشكلة الأصلية (من الأرشيف)

تم اكتشاف **6 جذور للمشكلة** في النظام القديم:

| # | المشكلة | المثال من الأرشيف | السبب الجذري |
|---|---------|-------------------|--------------|
| 1 | تضارب الحقول | `actor_v2` ≠ `war_data.actor` | 4 محركات تكتب فوق بعضها |
| 2 | الفاعل يُستنتج من المنطقة فقط | خبر عن العراق → إيران | ربط مفرط للكلمات الجيوسياسية |
| 3 | المصدر يلوّث القرار | "اليوم السابع" يصبح فاعل | عدم فصل المصدر عن الفاعل |
| 4 | نسخ الجملة كفاعل | "تجدد القصف..." يصبح الفاعل | غياب أنماط الاستخراج الدقيقة |
| 5 | حقول JSON مكسورة | `"hybrid_layers":["Array"]` | array to string conversion |
| 6 | غياب طبقة الثقة | نتائج ضعيفة تُحفظ كمؤكدة | لا يوجد نظام تقييم ثقة |

## ✅ الحل: V3 Core Rewrite

### المبادئ الأساسية

1. **مصدر واحد للحقيقة (Single Source of Truth)**
   - حقل واحد فقط: `actor_final`
   - كل الأنظمة الأخرى (`actor_v2`, `war_data.actor`, `field_data.actor_v2`, `who_primary`) تُشتق منه

2. **هرمية القرار (Decision Hierarchy)**
   ```
   المستوى 1 (95+ ثقة): ذكر صريح ← "حزب الله:"، "الجيش الإسرائيلي:"
   المستوى 2 (80-90 ثقة): استنتاج من الفعل ← "قصف لبنان" → إسرائيل
   المستوى 3 (0 ثقة): غير محسوم ← أخبار اقتصادية/سياسية بدون فاعل مباشر
   ```

3. **قائمة سوداء صارمة (Blacklist)**
   - ممنوع نهائيًا: "تم رصد"، "مصادر"، "اليوم السابع"، "قناة العربية"، "واشنطن بوست"...

4. **نظام ثقة شفاف**
   | الثقة | النتيجة | الإجراء |
   |-------|---------|---------|
   | 90+ | مؤكد | حفظ مباشر |
   | 70-89 | مرجح | مراجعة اختيارية |
   | 40-69 | ضعيف | يحتاج تحقق |
   | <40 | غير محسوم | وضع "غير محسوم" |

## 🔄 الترقية من V2 إلى V3

### قبل (V2 - مكسور)
```php
// 4 محركات متضاربة
$actor_v2 = $extractor->extract($text);        // قد يعطي "تم رصد"
$war_data['actor'] = $classifier->guess($text); // قد يعطي "إيران"
$field_data['actor_v2'] = $legacy->parse();     // قد يعطي الجملة كاملة
$who_primary = $regionMapper->map($region);     // قد يعطي "إيران" للعراق

// النتيجة: تضارب وفوضى
```

### بعد (V3 - موحد)
```php
// محرك واحد للقرار
$engine = new ActorDecisionEngineV3();
$result = $engine->decide($headline, $content, $region);

// حقل واحد نهائي
$actor_final = $result['actor'];
$confidence = $result['confidence'];

// كل الحقول الأخرى تُشتق من actor_final
$war_data['actor'] = $actor_final;
$field_data['actor_v2'] = $actor_final;
$who_primary = $actor_final;
```

## 🛠️ الإصلاحات المحددة للأرشيف

| رقم الحالة | قبل (خطأ) | بعد (صحيح) | آلية الإصلاح |
|-----------|-----------|------------|--------------|
| 1 | أين تذهب أموال نفط العراق؟ → إيران | فاعل غير محسوم | عدم الربط التلقائي بالمنطقة |
| 2 | تم رصد اقتحام... → "تم رصد اقتحام" | جيش العدو الإسرائيلي | القائمة السوداء + أنماط الفعل |
| 3 | اليوم السابع: ... → "اليوم السابع" | فاعل غير محسوم | القائمة السوداء للمصادر |
| 4 | تجدّد القصف... → الجملة كاملة | جيش العدو الإسرائيلي | استخراج الأنماط فقط |
| 5 | hybrid_layers: "Array" | ["عسكري","سياسي"] | دالة cleanForJson() |
| 6 | نتائج ضعيفة = مؤكدة | ثقة 0 = غير محسوم | نظام الثقة الهرمي |

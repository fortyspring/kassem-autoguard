## أمثلة على الاستخدام

### عبر WP-CLI:

```bash
# عد الأحداث الملوثة
wp so-cleanup-archive count

# عرض الإحصائيات
wp so-cleanup-archive stats

# تنظيف 50 حدث
wp so-cleanup-archive --limit=50

# تنظيف حتى 1000 حدث
wp so-cleanup-archive --max=1000

# تجربة دون تغييرات
wp so-cleanup-archive --dry-run
```

### عبر الواجهة الإدارية:

```
OSINT Events → تنظيف الأرشيف
```

### برمجياً:

```php
// عد الأحداث الملوثة
$dirty_count = so_count_dirty_events();

// تنظيف دفعة
$result = so_cleanup_archive_batch(50, 0);
echo "نجاح: {$result['success']}, فشل: {$result['failed']}";

// تنظيف حدث محدد
$single = so_clean_single_event($event_id);
if ($single['success']) {
    echo "الفاعل الجديد: {$single['new_actor']}";
}

// الحصول على الإحصائيات
$stats = so_get_cleanup_stats();
print_r($stats['top_actors']);
```

## أنواع التلوث المكتشفة

### 1. وكالات إعلامية
❌ **قبل:** "رويترز"، "العربية"، "الميادين"
✅ **بعد:** الفاعل الحقيقي المستنتج من السياق

### 2. أشخاص ومسؤولون
❌ **قبل:** "ترمب"، "وزير الخارجية التركي"، "رئيس البرلمان الإيراني"
✅ **بعد:** "الولايات المتحدة الأمريكية"، "تركيا"، "إيران"

### 3. دول فقط
❌ **قبل:** "فلسطين"، "إيران"، "أميركا"
✅ **بعد:** الجهة الفعلية (جيش، حكومة، فصائل)

### 4. مصادر عامة
❌ **قبل:** "مصدر أميركي"، "وكالة إيرنا"، "صحيفة يديعوت"
✅ **بعد:** الفاعل الحقيقي المذكور في الخبر

### 5. فاعلون غير محددون
❌ **قبل:** "فاعل غير محسوم"، "غير معروف"، "مجهول"
✅ **بعد:** الفاعل المستنتج بالمحرك الموحد

## معايير الاستبدال

لا يتم استبدال الفاعل إلا إذا:

1. ✅ الفاعل الحالي ملوث (according to is_junk_actor)
2. ✅ الفاعل الجديد ليس ملوثاً
3. ✅ مستوى الثقة > 50% (للحالات الحدية)
4. ✅ الفاعل الجديد أفضل بشكل واضح

## الأداء

- **حجم الدفعة الافتراضي:** 50 حدث
- **الحد الأقصى الموصى به:** 200 حدث/دفعة
- **الوقت المتوقع:** ~1-2 ثانية/حدث (يعتمد على التعقيد)
- **ذاكرة إضافية:** ~5MB لكل دفعة

## التوصيات

1. **تشغيل تدريجي:** ابدأ بـ 50 حدث واختبر النتائج
2. **مراجعة العينات:** افحص 10-20 حدث يدوياً قبل التشغيل الشامل
3. **النسخ الاحتياطي:** احتفظ بنسخة من قاعدة البيانات قبل البدء
4. **التوقيت:** شغّل في أوقات انخفاض الزيادة
5. **المراقبة:** راقب السجلات والإحصائيات أثناء التشغيل
